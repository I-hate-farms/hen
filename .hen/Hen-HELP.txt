Hen is a declarative build system for vala applications.

Here are the defined hen.sh commands: 
   - ./hen update: update the build system and overwrite the content of cmake/
   - ./hen prepare: generate the cmake files 
   - ./hen build: do everything to build the application
   - ./hen list: enumerates the make tasks that are defined
   - ./hen rebuild: recreate the build folder and build the application
